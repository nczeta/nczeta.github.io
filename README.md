# nczeta — Portfolio

Minimal dark-themed personal portfolio built from scratch with vanilla HTML, CSS and JavaScript. No frameworks, no build tools.

## Key Features

- **SVG loader** — custom logo drawn and erased on page load via stroke-dashoffset animation
- **Glassmorphism cards** — blur, amber glow on hover
- **Animated nav links** — letter-by-letter vertical flip on hover
- **Logo reveal** — two halves animate in from opposite directions via clip-path

## Tech Stack

- HTML5 / CSS3 / Vanilla JavaScript
- Bootstrap Icons, Font Awesome 6
- Google Fonts (Montserrat, Stoke)

## Usage

No install needed — just open `index.html` in a browser, or serve it locally:

```bash
npx serve .
```
